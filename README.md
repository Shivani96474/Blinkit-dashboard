Project Title:
Blinkit Sales Analytics Dashboard

Short Description:
An interactive dashboard that analyzes Blinkit sales data to track performance, customer preferences, and outlet-wise trends for data-driven decisions.

Tech Stack:
* Power BI
* DAX
* Microsoft Excel / CSV
* SQL 

Data Source:
Blinkit sales dataset (CSV files) containing item details, outlet information, sales, ratings, and categories.

Features:
* KPI tracking (Total Sales, Average Sales, Items, Ratings)
* Sales analysis by item type and fat content
* Outlet-wise performance (size, location, type)
* Year-wise outlet establishment trends
* Interactive filters and slicers
* Clean, user-friendly visualizations suitable for business insights# Blinkit-dashboard
